CeleroChallenge

Projeto de desenvolvimento de um jogo da velha utilizando a API fornecida pela Marvel.

Quick start

Clone o repositório:

git clone https://github.com/thoffart/CeleroChallenge.git

cd CeleroChallenge

Crie um arquivo config.ts na seguinte pasta src/app/config/ com suas chaves de API da marvel da seguinte maneira:

export const config = {
    marvel: {
        publicKey: 'YOUR_PUBLIC_KEY',
        privateKey: 'YOUR_PRIVATE_KEY'
    }
};

Após isso rode o projeto:

npm start

Tests

Para rodar os testes:

npm run test

Documentation

Para a documentação foi utilizado o Compodoc. Rode:

npm run compodoc
